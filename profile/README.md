## 🚀 Projeto em Destaque

### [Supply 2026](https://github.com/alantaru/SupplyV2) — Sistema de Gestão de Insumos de Impressão

[![Demo](https://img.shields.io/badge/🌐_Demo_ao_Vivo-supplyv2.duckdns.org-blue?style=for-the-badge)](https://your-domain.com)
[![Tests](https://img.shields.io/badge/Tests-1488_passing-4CAF50?style=for-the-badge&logo=pytest)](https://github.com/alantaru/SupplyV2/tree/main/backend/tests)
[![Coverage](https://img.shields.io/badge/Coverage-99%25-4CAF50?style=for-the-badge)](https://github.com/alantaru/SupplyV2)

Sistema SaaS completo desenvolvido para resolver problemas reais de gestão de contratos de impressão corporativa no Grupo Usiminas.

**O que foi construído:**
- 🖨️ Gestão de protocolos de entrega com wizard de 3 etapas
- 📦 Estoque inteligente com baixa automática ao registrar entrega
- 🗺️ Rotas logísticas com geração de protocolos em lote
- 📊 BI Dashboard executivo com 6 painéis analíticos
- 🔬 Importador universal de CSV/Excel com detecção automática de encoding
- 🔐 Multi-tenant com RBAC (user/admin/superadmin)

**Stack:** `Python 3.12` `FastAPI` `React 19` `AWS S3/EC2` `Playwright` `pytest` `Hypothesis`

**Qualidade:** 1488 testes | 99% coverage | 74 testes E2E | 0 falhas | CI/CD pipeline
